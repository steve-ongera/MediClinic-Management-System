System Overview
The Mediclinic Management System streamlines clinic operations with three user roles: Admin, Doctor, and Receptionist.
User Roles
Administrator
  • Financial Reports & Analytics
  • User Management
  • View All Consultations
  • System Configuration
Doctor
  • Patient Lookup
  • Create Consultations
  • Prescribe Medications
  • Generate Consultation Codes
Receptionist
  • Patient Registration
  • Appointment Scheduling
  • Payment Processing
  • Consultation Lookup
Login Process
  1. Access Login Page
    Navigate to the system login URL
  2. Enter Credentials
    Input your username and password
  3. Authentication
    System verifies credentials and identifies user type
  4. Redirect to Dashboard
    Based on user type (Admin/Doctor/Receptionist)
Patient Flow Process
Reception Process
  1. Patient Arrival
    Patient arrives at the clinic and approaches reception
  2. Check Registration
    Search patient in system by name or ID number
  3. New Patient Registration
    If new patient, collect:
    • Full name
    • ID/Passport number
    • Phone number
    • Email address
    • Physical address
  4. Queue Management
    Add patient to doctor's queue and notify when doctor is available
Doctor Consultation Process
  1. Patient Entry
    Doctor receives notification of next patient
  2. Patient Lookup
    Search for patient in consultation module
  3. Create Consultation
    • Record diagnosis
    • Add prescribed medications
    • Add any notes
  4. Generate Code
    System automatically generates unique consultation code
    Consultation Code: CLN-2023-05-001
Payment Process
  1. Patient Returns
    Patient returns to reception with consultation code
  2. Code Lookup
    Search for consultation using the provided code
  3. Process Payment
    System displays:
    • Consultation fee
    • Prescribed medications
    • Total amount due
  4. Completion
    System records payment and updates financial records
Administrator Features

Administrators can access comprehensive financial reports including:

  • Daily, weekly, monthly revenue
  • Payment type breakdown (cash, insurance, credit)
  • Service type revenue analysis
  • Outstanding payments
Use the date filters to analyze trends over specific periods.

Manage all system users with these capabilities:

Add New User
  1. Navigate to Users > Add New
  2. Fill in user details
  3. Assign appropriate role (Admin/Doctor/Receptionist)
  4. Set temporary password
  5. Save and notify user
Manage Permissions

Adjust user permissions as needed:

  • Enable/disable system access
  • Reset passwords
  • Audit login activity
Troubleshooting
Issue Solution
Can't find patient record
  • Check spelling variations
  • Search by ID number instead of name
  • Verify registration status
Consultation code not found
  • Verify code was entered correctly
  • Check if consultation was saved
  • Contact doctor for verification
System running slowly
  • Refresh the page
  • Clear browser cache
  • Report to IT if persistent
